# PF Lab Projects

A collection of C++ programs developed during the Programming Fundamentals course, covering game development and file handling concepts.

## 🎮 Projects Included

| Project             | Description                                      |
|---------------------|---------------------------------------------------|
| **Pacman Game**     | Classic Pacman-style game implemented in C++      |
| **Tic Tac Toe**     | Console-based Tic Tac Toe game                    |
| **Apni Game**       | Custom game project                               |
| **HMS File Handling** | Hospital Management System — file handling logic |
| **UAMS File Handling** | User/Account Management System — file handling |

## 🛠️ Tech Stack
- **Language:** C++
- **Concepts Covered:** OOP, file I/O, arrays, control structures, game logic

## ⚙️ How to Run
Compile any `.cpp` file using a C++ compiler (e.g., g++):
```bash
g++ ProjectName.cpp -o ProjectName
./ProjectName
```

## 🎓 About
Developed as part of the **Programming Fundamentals** course, BSCS program, University of Engineering and Technology (UET), Lahore — Faisalabad Campus.
